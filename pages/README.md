#  System Portfolio

A high-tech, multipage web application built with **Streamlit** and **Python**. This portfolio serves as a digital command center showcasing my projects and technical skills.

##  Key Features
- **High-Tech UI:** Custom CSS injection for a neon-glow.
- **Responsive Design:** Optimized for both desktop and mobile viewing.
- **Multipage Navigation:** Seamless transition between Home, About, Skills, and Projects.

## Tech Stack
- **Frontend:** Streamlit, CSS3 (Glassmorphism)
- **Backend:** Python
- **Database:** MySQL (used in featured projects)
- **Tools:** Git, VS Code, Streamlit Community Cloud

##  Project Structure
- `app.py`: Main entry point and system initialization.
- `pages/`: Individual modules for Home, About, Skills, and Projects.
- `.streamlit/`: Global theme configuration.
